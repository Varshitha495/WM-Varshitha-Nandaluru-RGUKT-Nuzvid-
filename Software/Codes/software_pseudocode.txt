START

Initialize system
Initialize Wi-Fi connection
Initialize ultrasonic sensor

WHILE system is powered ON
    Read distance value from ultrasonic sensor
    Convert distance into fill level percentage

    IF fill level percentage >= predefined threshold
        Mark bin status as "FULL"
    ELSE
        Mark bin status as "NOT FULL"
    END IF

    Send bin ID and fill level data to cloud server
    Wait for fixed time interval
END WHILE

STOP
